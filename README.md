# 记词宝 (Word Memorizer)

基于艾宾浩斯遗忘曲线的智能背单词神器。

## 核心功能
- **科学复习**：基于艾宾浩斯遗忘曲线自动安排 8 个复习阶段。
- **智能翻译**：集成翻译接口，支持四级词汇精准匹配。
- **多模式练习**：支持“英译中”、“中译英”和“拼写练习”。
- **自主背诵**：词卡浏览模式，支持遮挡释义，原位查看。
- **复习提醒**：侧边栏实时显示下一次背诵时间。
- **多账号系统**：支持多用户数据隔离，自定义头像和昵称。
- **可视化日历**：年、月、日三级视图，掌握记忆轨迹。

## 技术栈
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- Lucide Icons (图标)
- Date-fns (日期处理)

## 如何使用
直接在浏览器中打开 `index.html` 即可开始学习。
